# 🌾 India's Agriculture Crop Production Analysis

An interactive **Tableau Dashboard** developed to analyze India's agricultural crop production using district-level data. This project provides meaningful insights into crop cultivation, production, and geographical distribution through interactive visualizations.

---

## 📌 Project Overview

India's agriculture sector plays a vital role in the country's economy. This project leverages the **ICRISAT District Level Dataset** and Tableau to create an interactive dashboard that helps users explore crop production patterns across different states and districts.

The dashboard enables users to analyze agricultural data visually, making it easier to identify trends, compare production, and understand crop distribution across India.

---

## 🎯 Objectives

- Analyze agricultural crop production across India.
- Visualize district-wise and state-wise crop distribution.
- Compare cultivated area and production.
- Identify agricultural trends using interactive dashboards.
- Enable easy data exploration through filters and maps.
